# Release notes

App Store "What's New" copy. Paste into App Store Connect when submitting a new version.

## Submission checklist

For each new version page in ASC, before clicking **Add for Review**:

- [ ] Build selected (from TestFlight, after Apple finishes processing)
- [ ] "What's New in This Version" pasted from the section below
- [ ] **Version Release** set to **Automatically release this version** (Apple flips this back to manual by default each time — easy to miss)
- [ ] Export compliance / IDFA answers match the previous version's
- [ ] Screenshots / metadata unchanged (or updated if the UI changed materially)

## v1.0.1 (Build 20) — 2026-05-13

- The app is now called "Todos for Everyone"
- The sign-in page is fully translated into English, Chinese, Spanish, French, German, and Japanese
- Streamlined sign-in flow with quicker access to your profile and sign-out
- Visual polish and bug fixes

Thanks for using the app! Feedback welcome at the support email on the listing.

## v1.0 (Build 16) — 2026-05-13

First public release on the App Store.

- Sign in with Apple, Google, or email
- Capture todos with priority, due date, and category — all editable in place
- Custom categories with your choice of color and icon
- Cross-device sync when you sign in — pick up where you left off
- Available in English, Chinese, Spanish, French, German, and Japanese
- Dark mode, iPhone + iPad support
- Trash with 30-day recovery window
